# fate-grand-quiz.com AdSense 対策チェックリスト

Google AdSense のポリシー「パブリッシャーのコンテンツを含まない画面における広告の扱い」に基づき、  
本サイトが「コンテンツ不足」や「作成中ページ」と誤判定されないようにするためのチェックリスト。

---

## 🏠 1. トップページ（/）

**目的:** 初訪問者にサイトの概要と価値を伝える。

- [x] サイトのテーマ（例：「FGOサーヴァント愛を試すクイズプラットフォーム」）が文章で説明されている  
- [x] 画像のみでなく、テキスト本文が100文字以上存在する  
- [x] 「準備中」「Coming soon」などの表現が含まれない  
- [x] `<title>` と `<meta name="description">` に固有の内容が設定されている  
- [x] SSR/SSGでHTML内に本文テキストが含まれている（SPAのみで表示されない）

📄 **推奨リード文（約170字）**  
`Fate/Grand Order をもっと楽しむための非公式クイズサイトです。スキル効果やプロフィールからサーヴァントを推理し、知識を磨きながら最新イベントにも対応した問題で遊べます。1問ごとに解説付きなので、推しサーヴァントの新しい一面を発見しましょう。`

🧱 **構成サンプル**  
- `h1`: `FGOサーヴァント愛を試すクイズプラットフォーム`  
- リード文: 上記リードコピー  
- 価値訴求（例）：  
  - スキル・プロフィール・宝具クイズを網羅  
  - Atlas Academy API を利用した最新データ  
  - 全問に解説付きで知識の振り返りができる  
- CTA：`いますぐクイズを始める`

📝 **メタ情報例**  
- `<title>Fate/Grand Order クイズで推しをもっと知ろう | Fate-Grand-Quiz</title>`  
- `<meta name="description" content="FGOサーヴァントのスキル・プロフィール・宝具から推理する非公式クイズサイト。最新データを使った解説付き問題で、推しへの愛を試そう。">`

💡 **補足:**  
Next.jsの`generateStaticParams`や`export`を利用し、GooglebotがJS実行なしでも本文を取得できるようにする。

---

## 🧩 2. クイズ選択ページ（/quiz）

**目的:** 各クイズモードの概要を紹介する。

- [x] 各クイズに説明文がある（例：「スキルの効果だけでサーヴァントを推理」など）  
- [x] ボタンのみではなく、プレイ内容を理解できる文章がある  
- [x] 単なるナビゲーションページになっていない  
- [x] 主要クイズの説明がHTMLに出力されている（JS依存でない）

---

## 🎯 3. クイズプレイページ（/quiz/skill, /quiz/profile, /quiz/np）

**目的:** 設問・回答を通して体験型コンテンツを提供する。

- [x] 問題文・選択肢・解説がHTML上に存在する  
- [x] 「開始前」画面にも簡単な説明文がある  
- [x] 読み込み中・スタート画面などに広告を設置していない  
- [ ] 各モードに固有のタイトル・説明文が設定されている  

💡 **例:**  
`<title>スキルクイズ - FGOクイズ</title>`  
`<meta name="description" content="スキル効果だけでサーヴァントを当てよう。全5問の挑戦クイズ！">`

---

## 🏁 4. 結果ページ（/quiz/result）

**目的:** 結果と次への導線を提示。

- [ ] 「正答数」だけでなく、簡単な振り返り文やおすすめリンクがある  
- [ ] メタ情報（title/description）が設定されている  
- [ ] 固有のURLごとに異なる内容を提供している（同一ページ量産ではない）

---

## ⚠️ 5. 広告配置ルール

- [ ] コンテンツ（本文・問題文）を含むページにのみ広告を表示  
- [ ] ログイン、アラート、エラー、ボタンのみのページに広告を置かない  
- [ ] ページ上部に広告を置く場合、本文より上に配置しない  
- [ ] 作成中・準備中ページでは広告コードを貼らない  

---

## 🧠 6. 技術的対応（Next.js）

- [ ] 主要ページを `generateStaticParams` や `next export` で静的生成  
- [ ] `View Source` で確認し、初期HTML内に本文がある  
- [ ] クイズごとに固有OGP（`og:title`, `og:description`, `og:image`）を設定  
- [ ] SSG/SSR時に `metadata` API や `generateMetadata` を活用  

---

## 💬 7. 作成中・下書きページ

- [ ] “Coming soon” や “準備中” ページには広告を貼らない  
- [ ] 完成前ページには `<meta name="robots" content="noindex">` を設定  
- [ ] 公開前のルートは robots.txt で除外  

---
